# Kenichi Iida (quqjp) 👋

東京を拠点に活動するフリーランスのフロントエンドエンジニア／デザイナーです。  
2011年9月に独立し、スタートアップ企業のサービス開発に業務委託として携わっています。  
主に **React／TypeScript** を用いたモダンなフロントエンド実装を得意としています。  

個人開発では、Webアプリやモバイルアプリの企画・開発にも積極的に取り組んでおり、  
開発からデザイン、運用まで一人で完結させるスタイルを大切にしています。  

非常勤講師として、大学でアプリケーション開発を教えています。  
また、旅行や開発をテーマにした記事を [Note](https://note.com/quqjp) にて執筆中です。
